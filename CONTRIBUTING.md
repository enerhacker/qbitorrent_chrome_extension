# Contributing

Köszönöm, ha hozzájárulsz a projekthez.

## Hibajelentés

Hibajelentésnél érdemes megadni:

- böngésző neve és verziója
- bővítmény verziója
- qBittorrent verziója
- hiba pontos leírása
- képernyőkép, ha van
- lépések, amelyekkel a hiba előidézhető

## Fejlesztési javaslat

Fejlesztési javaslatnál írd le:

- mi a jelenlegi probléma
- milyen működést szeretnél
- milyen beállítást vagy felületet érintene

## Pull request

Pull request előtt érdemes ellenőrizni:

- a bővítmény betölthető Chrome-ban
- a `manifest.json` érvényes
- a beállítások oldal megnyílik
- a toast előnézet működik
- nincs felesleges ZIP vagy build fájl commitolva
