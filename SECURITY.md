# Security Policy

## Támogatott verziók

Jelenleg az aktuális legfrissebb verzió támogatott.

| Verzió | Támogatott |
| --- | --- |
| 1.3.x | Igen |
| 1.2.x és régebbi | Nem |

## Biztonsági hiba jelentése

Biztonsági hibát GitHub issue-ban vagy privát csatornán érdemes jelezni.

Kérlek, ne tegyél közzé nyilvánosan olyan részletes hibaleírást, amely más felhasználók rendszerét veszélyeztetheti.

## Fontos megjegyzés

A bővítmény qBittorrent WebUI belépési adatokat kezelhet, ezért csak megbízható környezetben használd, és ne oszd meg a saját beállításaidat tartalmazó fájlokat.
